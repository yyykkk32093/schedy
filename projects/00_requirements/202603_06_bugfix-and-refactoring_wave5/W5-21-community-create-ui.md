## 実装メモ

### [1] ブラウザバック修正
- 作成完了後の遷移を `navigate('/communities/' + id, { replace: true })` にする
- ヒストリスタックから作成画面を除外し、バック時にコミュニティ一覧へ戻る
- W5-26（アクティビティ作成/更新後ブラウザバック）と同パターン → 共通化可能

### [2] 活動頻度「指定なし」見切れ
- Step2 の活動頻度 Select コンポーネントの幅を確認
- テキストが途切れている場合は `min-width` or `truncate` + tooltip で対応

### [3] 活動拠点入力追加
- CommunitySettingsPage の活動拠点入力UIを確認
  - テキスト入力? 地域マスタからの選択?
- 同じ入力パターンを CommunityCreatePage Step2（or Step1）に追加
- バックエンド: CreateCommunityUseCase に活動拠点フィールドが既に存在するか確認が必要

### [4][5] 年齢プルダウン連動
- ageMin Select → 選択値以上の選択肢のみ ageMax に表示（+「指定なし」）
- ageMax Select → 選択値以下の選択肢のみ ageMin に表示（+「指定なし」）
- いずれかが「指定なし」の場合は他方のフィルタリング不要

### [6] 推奨レベル「指定なし」+ カテゴリ必須
- 現在 Slider（0〜8）→「指定なし」の表現方法:
  - チェックボックス「レベル指定なし」を追加し、ON 時 Slider 非活性?
  - Slider の左端を「指定なし」にする?
- カテゴリ: バリデーションで必須に変更（`required` 属性追加 or フォームバリデーション）
